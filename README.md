# Academy_web
Academia de Inglés - Sitio Web
Este proyecto es una aplicación web desarrollada para una academia de inglés.
Permite gestionar usuarios, publicar noticias y administrar citas, con una interfaz diferenciada para alumnos y administradores.
Funcionalidades principales:
  Gestión de noticias - Los daministradores puedes publicar y editar noticias visibles para todos los usuarios.
  Sistema de citas - Los usuarios puedes visualizar y solicitar citas, mientras que los administradores las gestionan.

Autenticación de usuarios:
  Login como usuario - acceso al perfil personal y a citas.
  Login como administrador - acceso a panel de gestión de usuarios, noticias y citas.
  Panel de administración - interdas privada para gestionar el contenido y los usuarios registrados.

Tecnologías utilizadas
  Frontend - HTML, CSS, JavaScript
  Backend - PHP
  Base de datos - MySQL
  Hosting gratuito - InfinityFree
